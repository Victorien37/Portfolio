Présentation du contexte

Le but de ce stage était de créer un add'on pour le logiciel de rendu 3D Blender en utilisant son API (Python)
L'add'on est un moteur de jeu qui premet de se déplacer dans un environnement en 3 dimentions non prédictible. Le but était d'automatiser un personnage qui marche devant, dérrière, à gauche, à droite de lui tout en gérant les collisions, les pentes, ...
